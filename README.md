# go-http-crud
API HTTP CRUD em Go para gerenciamento de contactos em memória.

- Implementa um servidor RESTful simples usando apenas pacotes padrão (net/http, encoding/json, etc.)
- Suporta GET, POST, PUT e DELETE no endpoint /contacts
- Utiliza um map[int]Contact para armazenar os contactos por id
- Ideal para aprender a criar APIs básicas em Go sem dependências externas
